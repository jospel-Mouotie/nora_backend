# Commandes pour créer le projet React Native

## 🚀 Méthode la plus simple : CMD

### 1. Ouvrir l'invite de commandes
- Appuyer sur `Win + R`
- Taper `cmd` et appuyer sur Entrée

### 2. Exécuter ces commandes une par une :

```cmd
cd /d c:\Users\jospel\Nora-v2
```

```cmd
npx react-native init NoraMobile --template react-native-template-typescript
```

```cmd
cd NoraMobile
```

```cmd
npm install @react-navigation/native @react-navigation/stack @react-navigation/bottom-tabs
```

```cmd
npm install @reduxjs/toolkit react-redux
```

```cmd
npm install axios react-native-vector-icons
```

```cmd
npm install react-native-gesture-handler react-native-reanimated
```

```cmd
npm install react-native-safe-area-context react-native-screens
```

```cmd
npm install @react-native-async-storage/async-storage
```

### 3. Vérifier que le projet est créé
```cmd
dir
```

Vous devriez voir le dossier `NoraMobile` créé avec tous les fichiers du projet React Native.

---

## 📱 Après la création du projet

Une fois que le projet est créé avec succès, nous pourrons :

1. **Créer la structure des dossiers** détaillée dans les fichiers précédents
2. **Commencer par l'écran d'onboarding** avec tes 4 images
3. **Développer l'écran de sélection des centres d'intérêt**
4. **Implémenter le Home Screen** avec produits recommandés

Le projet sera dans `c:\Users\jospel\Nora-v2\NoraMobile` et nous aurons une vue complète sur le backend et le frontend ! 🎯
